公式

  • 1.圆类
  • 2.棱柱
  • 3.锥体

圆类

圆

r(R): 半径; d(D): 直径;

周长：C = 2πr = πd

面积: S = πr²

半圆周长: C = πr+2r

半圆面积: S = πr²/2

扇形

扇形弧长：L = 圆心角（弧度制） * r = n°πr/180°（n为圆心角）

扇形面积：S =n π r²/360=Lr/2（L为扇形的弧长）

球

球体积： V = (3/4)πr³

球表面积: S = 4πr² = πd²

圆锥

L: 母线长;r: 底面半径

表面积: S = πr²+πrl

侧面积: S = πrl

体积: V = 1/3Sh（S是底面积,h是圆锥高）

圆锥底面半径: r = n°/360°L

圆柱

圆柱体积： V = sh

圆柱体积：V = 底面积×高 或 V=1/2侧面积×高

圆柱侧面积：S = ch = 2πrh = πdh

圆柱侧面积：S侧 = 底面周长×高

圆柱表面积：s = ch+2πr²

圆柱表面积：S表 = 侧面积+2个底面积

棱柱

通用公式:

体积: V = S底 * 高

侧面积: S = 底面周长 * 高

表面积: S = S侧 + S底 * 2

圆柱

圆柱体积： V = sh

圆柱体积：V = 底面积×高 或 V=1/2侧面积×高

圆柱侧面积：S = ch = 2πrh = πdh

圆柱侧面积：S侧 = 底面周长×高

圆柱表面积：s = ch+2πr²

圆柱表面积：S表 = 侧面积+2个底面积

锥体

圆锥

L: 母线长;r: 底面半径

表面积: S = πr²+πrl

侧面积: S = πrl

体积: V = 1/3 S底 * h

圆锥底面半径: r = n°/360°L
