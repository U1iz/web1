当 D²+E²-4F > 0 时:

圆心: (-D/2,-E/2)

一般方程仅表示一个点 (-D/2,-E/2) ，叫做 点圆(半径为零的圆)

当D²+E²-4F < 0 时:

没有一个点的坐标满足圆的一般方程，即一般方程不表示任何图形，叫做 虚圆

直线方程

斜率 (k)     Baidu

斜率公式: k = tanα (α ≠ π/2)

斜率公式:k = (y₂-y₁) / (x₂-x₁) (x₁ ≠ x₂)

设直线方程 Ax + By + C = 0

斜率公式: k = -(A/B)

设 y = k₁*x+b 和 y = k₂*x + b 若斜率存在， k₁ = k₂ 则两直线 平行

设 y = k₁*x+b 和 y = k₂*x + b 若斜率存在， k₁*k₂ = -1 则两直线 垂直
