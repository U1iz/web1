等差数列

定义
等差数列的定义： 一般地，如果一个数列从第2项起，每一项与它的前一项的差等于同一个常数，那么这个数列就叫做等差数列，这个常数叫做公差，用符号语言表示为aₙ₊₁ - aₙ = d
公差

公差: d = (aₙ-a₁)/(n-1) (n ≥ 2，n ∈ Z+)

公差 = (末项 - 首项) ÷（项数-1）（其中n大于或等于2，n属于正整数）

项数

项数: n =(aₙ - a₁)/d + 1；

项数 =（末项-首项）÷ 公差 + 1；

末项

末项: aₙ = a₁ + (n - 1) * d

末项 = 首项 +（项数-1）× 公差

首项

首项: a₁ = aₙ - (n - 1) * d

首项 = 末项 -（项数-1）× 公差

求和

求和: Sₙ = (a₁ + aₙ) * n/2

求和：（首项+末项）× 项数/2

前n项的和

前n项的和: Sₙ = (a₁ * n ) + [n * (n-1)d] / 2

前n项的和Sₙ = 首项 × n + （ 项数 x (项数-1）公差）÷ 2 （已知首项与公差）

前n项的和: Sₙ = n(a₁ + aₙ) / 2

前n项的和Sₙ = 项数 x （首项 + 末项）÷ 2 （已知首项与末项）

等差中项

若a, A, c 成等差数列, 则 A = (a + b) / 2

当n为偶数时，等差中项为中间两项，这两项的和等于首尾两项和，等于二倍的总和除以项数n

数列为奇数项时: Sₙ = 中间一项×项数

Sₙ = b * n (中项用b代替)

数列为偶数项时: Sₙ = 中间两项和×项数的一半

Sₙ = (b₁ + b₂) * n/2 (中项用b代替)

等比数列

定义
等比数列的定义： 一般地，如果一个数列从第2项起，每一项与它的前一项的比等于同一个常数，那么这个数列就叫做等比数列，这个常数叫做公比，用符号语言表示为aₙ₊₁ / aₙ = q
通项公式:

通项 aₙ = a₁ * q⁽ⁿ⁻¹⁾

通项：末项 = 首项 x 公比的n-1次方

求和公式

求和1: Sₙ = a₁(1-qⁿ) / (1-q) (q ≠ 1)

求和2: 和 = 首项（1 - 公比的数列项数次方）÷ （1 - 公比）

求和2: Sₙ = (a₁ - aₙ*q) / (1-q) (q ≠ 1)

求和2：和 = （首项 - 末项 x 公比） ÷ （1-公比）

注意：同号的两个数才有等比中项，并且它们的等比中项有两个（两个等比中项互为相反数）
